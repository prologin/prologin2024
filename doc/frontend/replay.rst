======
Replay
======

Lecture
-------

La lecture du replay se fait via les contrôles (`↞`, `←`, `∥`, `→`, `↠`).

Keybinds
-------

| Les contrôles du replay sont les suivants:
| `flèche gauche` : Action précédente
| `flèche droite` :  Action suivante
| `flèche bas` : Début
| `flèche haut` : Fin
| `espace` : Affichage automatique / pause
